# Treoir an tSuímh

Úsáideann an suíomh seo **SvelteKit**, **Tailwind CSS**, agus **mdsvex** chun
stádas cluichí a fhoilsiú mar shuíomh statach.

## Conas a oibríonn sé

- Léitear sonraí ó `static/status.json`
- Taispeántar cluichí ar an leathanach baile agus ar leathanaigh mhionsonraithe
- Is féidir le leathanaigh Markdown breise a bheith ann do nótaí agus treoracha

## Cén fáth mdsvex?

Ligeann sé duit Markdown a úsáid taobh istigh de shreabhadh SvelteKit gan
seachbhóithre neamhaibí ná córais inneachair ró-throm.
